# Roadmap

## Built Now

- Local Autody liquidity dashboard
- AU/USDT planning calculator
- Price impact simulator
- Polygon token reader
- Uniswap v3 Polygon pool discovery
- VPS monitor worker
- Deployment guide

## Next Engineering Layer

- Add QuickSwap v3 Algebra pool address config after the AU/USDT pool exists.
- Add candle storage in SQLite.
- Add holder and transfer monitoring.
- Add Telegram or email alerts.
- Add password-protected owner portal.
- Add wallet-connect transaction review.
- Add LP position reader for NonfungiblePositionManager NFTs.

## Not Included

- Private key storage
- Automatic swap execution
- Artificial volume generation
- Fake activity tools

The system should help manage real liquidity and visibility without creating public-market manipulation risk.
